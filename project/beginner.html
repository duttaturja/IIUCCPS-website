Course Outline

  • Duration: 4 months with Track-sheet activities and guidance till semester ends.
  • Total Session: 15
  • Registration fees: 1200
  • Syllabus:
    • Session 1: Introduction to time and space complexity and online judges
    • Session 2: Array and Vector
    • Session 3: Binary Search
    • Session 4: Introduction to JTS and problem tags

    • Session 9: Combinatorics
    • Session 10: Binary Search (Codeforces Edu)
    • Session 11: Recursion
    • Session 12: Computational Geometry,Vector / Point
    • Session 13: DFS
    • Session 14: BFS
    • Session 15: Basic Game theory, Two pointers, Basic Bit Manipulation
